# MERN Ecommerce

Full Stack Ecommerce Application built using MERN Stack.

## Tech Stack

Frontend
- React
- Tailwind CSS
- React Router

Backend
- Node.js
- Express
- MongoDB

## Features

- Authentication (Login / Signup)
- Cart System
- Checkout
- Order Success Page
- Profile Page
- Responsive UI
- Mobile Bottom Navigation

## Run Locally

### Client

cd client
npm install
npm run dev

### Server

cd server
npm install
npm start
